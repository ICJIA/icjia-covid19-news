# ICJIA News & Updates

## Demo

https://icjia.illinois.gov/covid19

## Install

```
$ git clone https://github.com/ICJIA/icjia-covid19-news.git
$ cd icjia-covid19-news
$ npm install
```

## Develop

```
npm run serve
```

## Build

```
npm run build
```

## Deploy

Publish directory: `./dist`
